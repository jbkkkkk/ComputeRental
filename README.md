## Реализация
- Выполнены требования к доступным действиям API
- Сервисы покрыты unit-тестами
- Добавлены логи
- Данные хранятся в PostgreSQL
- Проблема конкурентных запросов решена с помощью оптимистической конкуренции
- Арендованный сервер выключается автоматически через 20 минут после выдачи

## Запуск
1. Клонировать репозиторий
2. В директории с исходным кодом выполнить `docker-compose up`
3. Применить миграции из той же директории `dotnet ef database update --project src/Infrastructure --startup-project src/API `
4. Swagger доступен по пути `http://localhost:5000/swagger`

## Разработка
